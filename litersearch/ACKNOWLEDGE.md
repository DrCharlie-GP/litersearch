# 致谢 / Acknowledgements

## 项目灵感来源

本项目 **litersearch** 的核心架构和设计理念深受 [evil-read-arxiv](https://github.com/yourusername/evil-read-arxiv) 项目的启发。evil-read-arxiv 是一套优秀的 Claude Code 技能集合，专注于 arXiv 论文的自动化搜索、推荐和分析工作流。

## 借鉴的核心架构

### 1. **技能组织结构**
- **目录结构**：采用 `scripts/` 目录存放 Python 脚本，`SKILL.md` 作为技能定义文件
- **模块化设计**：将检索、分析、笔记生成等功能拆分为独立模块
- **配置管理**：通过 YAML 配置文件管理研究领域、关键词、API 密钥等

### 2. **工作流设计**
- **每日推荐流程**：借鉴 `start-my-day` 的工作流，实现 `med-start-my-day`
  - 自动检索最新文献
  - 多维度评分排序
  - 生成每日推荐笔记
  - 为高优先级论文生成详细笔记
- **关键词链接**：借鉴 `link_keywords.py` 和 `scan_existing_notes.py` 的思路
  - 扫描现有笔记构建索引
  - 自动将关键词链接到已有笔记

### 3. **评分系统**
- **多维度评分**：参考 evil-read-arxiv 的评分机制，设计了医学文献专用的评分系统
  - 数据源权重（30%）
  - 期刊影响力（35%）
  - 研究类型（20%）
  - 引用量（15%）
  - 特殊加成（中国相关、定性研究等）

### 4. **笔记生成**
- **结构化笔记模板**：借鉴 `paper-analyze` 的笔记结构
  - 基本信息（作者、期刊、年份、DOI/PMID）
  - 摘要（中英文双语）
  - 研究笔记（目的、方法、发现、意义、局限性）
  - 个人评论
  - 相关文献

## 针对医学文献的定制与优化

### 1. **医学专业数据源**
- **PubMed 集成**：添加 PubMed API 支持，这是医学文献的权威数据库
- **MeSH 术语映射**：实现 40+ 医学概念到 MeSH 术语的自动映射
- **专业检索式构建**：自动构建符合 PubMed 语法的检索式（MeSH + 文本词 + 日期过滤）
- **多源并行检索**：整合 PubMed、Scopus、Semantic Scholar、OpenAlex

### 2. **医学文献评分系统**
- **期刊影响力数据库**：内置 100+ 顶级医学期刊评分（NEJM、Lancet、JAMA 等）
- **研究类型识别**：自动识别 RCT、Cohort、Review、Qualitative 等研究类型
- **中国相关研究标记**：识别中国作者和研究场景，给予额外加分
- **定性研究支持**：特别关注定性研究方法

### 3. **主题分析功能**
- **Claude API 集成**：使用 Claude API 进行智能主题分析
  - 从摘要中提取研究内容（15-30字概括）
  - 聚类相似研究到主题和研究焦点
  - 为每条研究内容匹配最相关的 3 篇文献
- **双层结构报告**：主题 → 研究焦点 → 具体研究内容 → 文献链接
- **Obsidian 内部链接**：自动生成 `[[论文标题|作者 年份]]` 格式的链接

### 4. **翻译功能**
- **自动摘要翻译**：默认使用 Claude API 翻译英文摘要为中文
- **自定义 API 支持**：兼容 OpenAI 格式的自定义翻译 API
- **批量翻译工具**：提供独立的翻译脚本，可翻译现有笔记

### 5. **大规模文献综述支持**
- **CSV 导出**：支持导出检索结果为 CSV 格式，便于 Excel 筛选
- **可选评分系统**：大规模检索时可关闭评分系统以提高速度
- **批量处理优化**：
  - 分批获取文献详情（200篇/批）
  - API 速率限制控制（有 API key 时 10次/秒）
  - 避免 HTTP 414 错误

### 6. **布尔运算符支持**
- **NOT 运算符**：支持排除特定关键词（如排除动物研究、细胞研究）
- **复杂检索式**：支持 `dementia care NOT (animal OR mouse OR rat)` 等复杂查询
- **日期范围灵活性**：支持多种日期格式（5y、90d、2026/01/01:2026/03/14）

### 7. **参考文档**
- **PubMed 检索规范指南**：详细说明 PubMed 检索语法和最佳实践
- **MeSH 术语参考表**：提供常用医学主题词表
- **文献处理配置**：可自定义期刊评分和研究类型权重

## 主要差异总结

| 特性 | evil-read-arxiv | litersearch |
|------|----------------|-------------|
| **目标领域** | 计算机科学（arXiv） | 医学与健康科学 |
| **数据源** | arXiv + Semantic Scholar | PubMed + Scopus + Semantic Scholar + OpenAlex |
| **检索语法** | arXiv 分类 + 关键词 | MeSH 术语 + 布尔运算符 + 日期过滤 |
| **评分维度** | 相关性 + 新近性 + 热门度 + 质量 | 数据源 + 期刊影响力 + 研究类型 + 引用量 |
| **特色功能** | 图片提取、顶会论文搜索 | 主题分析、摘要翻译、CSV 导出 |
| **笔记标识** | arXiv ID | PMID + DOI |
| **目标用户** | AI/ML 研究者 | 医学研究者、临床医生、公共卫生学者 |
